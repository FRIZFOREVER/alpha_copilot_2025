# Функциональные требования по фронтенду

## AI Copilot для малого бизнеса

### Основные требования

1. **Мобильный/мессенджер UX**: Полностью адаптивный дизайн, быстрый запуск (<2 сек), голосовой ввод, минимальное количество кликов.

2. **Система чатов**: Множественные чаты с тегами (финансы, маркетинг, документы, юридические), автоматическое создание, поиск по истории, группировка по категориям.

3. **Memories / Context RAG**: Автоматическое проставление тегов, визуализация связанного контекста, суммаризация диалогов, поиск по базе знаний через Qdrant, отображение связанных сообщений.

4. **Персонализация**: Расширенный профиль с бизнес-контекстом (тип бизнеса, индустрия, размер), интеграции (банк, CRM, витрины), использование контекста в ответах AI.

5. **Работа с документами**: Загрузка (PDF, DOCX, XLSX), вопросы по документам, генерация документов по шаблонам, суммаризация, экспорт.

6. **Маркетинговый Copilot**: Генерация постов для соцсетей, создание промо-механик, email-рассылки, аналитика контента, календарь публикаций.

7. **Агентная система поиска**: Автоматический веб-поиск с индикаторами, отображение источников, настройки поиска.

8. **Google MCP интеграция**: Calendar (планирование), Drive (документы), Gmail (письма), Sheets (таблицы), Docs (редактирование).

9. **Финансовая аналитика**: Интеграция с Альфа-Банк API, балансы счетов, анализ транзакций, категоризация расходов, прогнозы.

10. **Юридические вопросы**: Консультации, генерация договоров, анализ документов, шаблоны, чек-листы.

11. **Проактивная аналитика**: Умные рекомендации, анализ операционных данных, уведомления, прогнозирование трендов.

### Приоритизация: Фаза 1 (MVP) - чат, профиль, документы, маркетинг. Фаза 2 - Memories/RAG, банк, юридические. Фаза 3 - Google MCP, агентный поиск.
